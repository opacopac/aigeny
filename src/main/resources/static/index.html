🔑 Jira Personal Access Token

Gib deinen persönlichen Jira-Token ein.
Wo findest du den Token?
Jira öffnen → Profil (oben rechts) → Personal Access Tokens → Token erstellen.

🔑 Bitbucket Personal Access Token

Gib deinen persönlichen Bitbucket-Token ein.
Wo findest du den Token?
Bitbucket öffnen → Profil (oben rechts) → Personal access tokens → Token erstellen (nur Lese-Rechte nötig).
